# TERRAFORM_KVM_W2025
Terraform script for bulk Windows 2025 server creation with KVM

Please make sure to edit the script where needed and use a sysprepped W2025 image with virtio drivers installed and network and hard drive using virtio (not sata) bus. These drivers are installed from virtio driver iso while installing Windows 2025: https://fedorapeople.org/groups/virt/virtio-win/direct-downloads/archive-virtio/?C=M;O=D.
